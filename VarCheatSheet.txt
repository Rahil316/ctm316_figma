Working with variables

On a ‚Ā†Variable‚Ā†:

- Set value for a mode:
‚Ā†variable.setValueForMode(modeId, value)‚Ā†
(value type depends on variable type: string, number, {r,g,b,a}, boolean, alias, etc.)

- Get values by mode (base collection only):
‚Ā†variable.valuesByMode‚Ā†

- Get values by mode for a specific (possibly extended) collection (Enterprise):
‚Ā†await variable.valuesByModeForCollectionAsync(collection)‚Ā†

- Resolve value for a consumer (node/style):
‚Ā†variable.resolveForConsumer(consumer)‚Ā†

Binding variables to nodes and styles

Generic binding via ‚Ā†setBoundVariable‚Ā†

On many nodes and styles:

- Bind variable to a simple property:
‚Ā†nodeOrStyle.setBoundVariable(field, variableOrNull)‚Ā†Example:
‚Ā†rect.setBoundVariable("width", widthVariable)‚Ā†
‚Ā†rect.setBoundVariable("width", null)‚Ā† // unbind

- Read bound variables map:
‚Ā†nodeOrStyle.boundVariables‚Ā†
Example access: ‚Ā†node.boundVariables["width"]‚Ā†

Typography-specific on ‚Ā†TextNode‚Ā† / ‚Ā†TextStyle‚Ā†

Bindable fields: ‚Ā†fontFamily‚Ā†, ‚Ā†fontStyle‚Ā†, ‚Ā†fontWeight‚Ā†, ‚Ā†lineHeight‚Ā†, ‚Ā†letterSpacing‚Ā†, ‚Ā†paragraphSpacing‚Ā†, ‚Ā†paragraphIndent‚Ā†.

On ‚Ā†TextNode‚Ā†:

‚Ā†textNode.setBoundVariable(field, variableOrNull)‚Ā†

- Bind to a range (substring):

- Get bound variables for whole node:
‚Ā†textNode.boundVariables‚Ā†

- Get bound variable for a range:
‚Ā†textNode.getRangeBoundVariable(start, end, field)‚Ā†

- Inspect styled segments with bound variables:

On ‚Ā†TextStyle‚Ā†:

- Bind typography variable:
‚Ā†textStyle.setBoundVariable(field, variableOrNull)‚Ā†

- Read bound variables:
‚Ā†textStyle.boundVariables‚Ā†

On ‚Ā†figma.variables‚Ā†:
